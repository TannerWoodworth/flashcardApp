## Installation instructions

First, be sure that Node is installed on your computer.
	
		node -v

Make sure Gulp is globally installed, by running:

    npm install -g gulp

After cloning the project, run the following commands:

    npm install
    gulp
    gulp dev
   
npm may need sudo
